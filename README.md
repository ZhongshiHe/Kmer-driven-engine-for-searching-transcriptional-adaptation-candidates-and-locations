# Kmer-driven-engine-for-searching-transcriptional-adaptation-candidates-and-locations
Code source of the manuscript: 

UPF1-dependent transcriptional adaptation mediates genetic compensation resulting from disruption of a glycerol-3-phosphate acyltransferase gene in Chlamydomonas reinhardtii

Manuscript authors: Yuting Lin (first author), Zhongshi He, Yantao Li (corresponding author) 

# Usage of run_pure_hamming_mapper.py
python run_pure_hamming_mapper.py -k 19 -i 80%

# Manual: 
-k: kmer length, -i: identity. 
The size of kmer (-k) represents the length of sRNA (generated by the Nonsense-mediated Decay); 

-k 19 and -i 80% combination means that hamming distance mapping endures 3 single bases mismatches, which accomodates the sRNA pairing mechanisms;

Usages caution: please place the gDNA_withUTR.fasta and transcripts_renamed.fa in the same directory with run_pure_hamming_mapper.py.
